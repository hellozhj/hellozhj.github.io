盒子布局相关

1. 盒子內减模式 box-sizing（神）

*{
 	box-sizing: border-box; 
}

标准(外增)盒子模型：最终实体范围是宽高+内边距+边框

内减盒子模型：最终实体范围就是 宽高 ，而内边距和边框是在宽高基础上收缩进盒子里面

2. 盒子宽度的计算 – calc 函数

在声明CSS属性值时执行一些计算(常用于计算可自适应的宽高)

 width: calc(100% - 80px);

可以+ - * /计算,但 + - 需要前后空格

双飞翼布局:版心和屏幕不固定大小,左右固定宽度,中间宽度自适应

/* 颜色后面加一个百分比就是颜色开始渲染的位置,加两个百分比就一个开始一个结束 */
background: linear-gradient(180deg, tomato 0% 20%, royalblue 20% 80%, orange 80% 100%);

过渡（transition）

  1. 执行过渡的属性名:transition-property

    如width，height 写all代表全部

  2. 执行过渡的持续的时间:transition-duration

    transition-duration:10s

  3. 速度曲线:transition-timing-function

    • linear： 匀速

    • ease： 慢-快-慢 默认值

    • ease-in： 慢-快。

    • ease-out： 快-慢。

    • ease-in-out： 慢-快-慢。

    • steps 设置 跳跃性的动画

      steps (n) 过渡分成n段

4.延迟时间:transition-delay

复合写法：

/* 过渡的属性为width 持续3s 匀速 延迟0s */
  transition: width 3s linear 0s;

多个过渡写法：

transition: 
  width 1s ease-in 1s,
  height 10s ease-in-out 2s;

2D转换（掌握）transform

2d移动 translate

正数是右下 负数是左上 可以设置百分比或者像素值

可以改变元素在页面中的位置，类似 定位

语法： translate(x,y)

百分比单位是相对于自身元素的 translate:(50%,50%);

translate类似定位，不会影响到其他元素的位置

对行内标签没有效果

2d旋转 rotate

语法:

transform: rotate(0deg);
  1. 角度为正时 顺时针 负时 为逆时针
  2. 默认旋转的中心点是元素的中心点

2d缩放 scale

scale(宽的倍数,高的倍数)

只写一个参数 第二个参数则和第一个参数一样

transform:scale(0.5,0.5) 缩小

transform:scale(-2,-2) 反向放大2倍

转换中心 transform-origin （了解）

  1. transform-origin:50% 50%; 默认值 元素的中心位置 百分比是相对于自身的宽度和高度
  2. transform-origin:top left; 左上角 和 transform-origin：0 0;相同
  3. transform-origin:50px 50px; 距离左上角 50px 50px 的位置
  4. transform-origin：0; 只写一个值的时候 第二个值默认为 50%;

3D转换（变换）

做三维的物体基本都要加 透视属性 和 转换方式属性

3d转换 无论旋转 缩放 位移 都可以指定某个轴向

rotate = rotateX + rotateY + rotateZ
translate = transtlateX + transtlateY + transtlatez

3d移动 translate3d

  1. transform: translate3d(x,y,z) 其中 x y z 分别指要移动的轴的方向的距离
  2. translform: translateX(100px) 仅仅是移动在x轴上移动
  3. translform: translateY(100px) 仅仅是移动在Y轴上移动
  4. translform: translateZ(100px) 仅仅是移动在Z轴上移动

z轴是垂直屏幕,默认是看不到元素在z轴的方向上移动，想要看到，可以使用的 视距 属性

视距(透视) perspective

物体 在z轴上的移动 的 远大近小 效果时

设置给要观察元素的父元素上

调整的是镜头相对于页面的视角

perspective: 1000px;
视距原点 perspective-origin（了解）
  1. 视距原点和视距一样，也是设置给要观察元素的父元素上
  2. perspective-origin:center center; 默认值是元素的中心点
  3. perspective-origin:10px； 指定了一个参数的时候，第二个参数默认为center 也就是50%；
  4. perspective-origin:10% %； 百分比都是相对于自身的宽度和高度

3d旋转 rotate3d

  1. transform:rotateX(45deg); 沿着x轴正方向旋转 45度
  2. transform:rotateY(45deg) 沿着y轴正方向旋转 45deg
  3. transform:rotateZ(45deg) 沿着Z轴正方向旋转 45deg
  4. transform:rotate3d(x,y,z,deg) 沿着自定义轴旋转 deg为角度 了解

3D缩放 scale3d （了解）

  1. transform: scale3d(1 ,1,2); 宽，高 缩放一倍，厚度放大两倍
  2. transform: scaleX(1) 只缩放宽
  3. transform: scaleY(1) 只缩放高
  4. transform: scaleZ(1) 只缩放厚

转换样式 transform-style

控制子元素是否开启3维立体环境**(添加给父元素)**

transform-style: preserve-3d; 3维立体环境

transform-style: flat; 平面模式 - 不开启3维立体环境


动画 animation（掌握）

动画可以自动播放可以设置变化的次数 甚至是无数次 

/* 1 声明动画函数 */

@keyframes ani_div {
  0%{
    width: 100px;
    background-color: red;
  }
  50%{
    width: 150px;
    background-color: green;
  }
  100%{
    width: 300px;
    height: 300px;
    background-color: yellow;
  }
}

div {
  width: 200px;
  height: 200px;
  background-color: aqua;
  margin: 100px auto;
  /* 2 调用动画 */
  animation-name: ani_div;
  /* 持续时间 */
  animation-duration: 2s;
}

语法：

  1. 动画名

    设置要使用的动画名 animation-name:xxx;

  2. 持续时间

    设置动画播放的持续时间 animation-duration:3s

  3. 速度曲线 linear匀速

    和设置过渡的速度曲线一样 animation-timing-function:linear;

    • linear： 匀速

    • ease： 慢-快-慢 默认值

    • ease-in： 慢-快。

    • ease-out： 快-慢。

    • ease-in-out： 慢-快-慢。

    • steps 设置 跳跃性的动画

      steps (n,start) 在该段时间的开始 触发

      steps(n,end) 在该段时间的 末端 出发

  4. 延迟时间

    animation-delay: 0s;

  5. 循环次数

    设置动画播放的循环次数 animation-iteration-count: 2; infinite 为无限循环

  6. 循环方向

    animation-direction

    如在动画中定义了 0%：红色 100%：黑色 那么 当属性值为

    1. normal 默认值 红 -> 黑
    2. reverse 反向运行 黑 -> 红
    3. alternate 正-反-正… 红 -> 黑 -> 红…
    4. alternate-reverse 反-正-反.. 黑 -> 红 -> 黑 …
    5. 以上与循环次数有关

  7. 动画等待或者结束的状态

    animation-fill-mode 设置动画在等待或者结束的时候的状态

    • forwards：动画结束后，元素样式停留在 100% 的样式
    • backwards： 在延迟等待的时间内，元素样式停留在 0% 的样式
    • both： 同时设置了 forwards和backwards两个属性值

  8. 暂停和播放

    animation-play-state 控制 播放 还是 暂停

    running 播放 paused 暂停

    复合写法

    animation: name duration timing-function delay iteration-count direction fill-mode;

    多个动画写法

animation:
name duration timing-function delay iteration-count direction fill-mode，
animation: name duration timing-function delay iteration-count direction fill-mode;

弹性布局

注意：

  1. 一排显示，如果子盒子加了百分比，就按百分比显示，如果需要盒子间有空隙，就需要宽度calc来计算。
  2. 如果父盒子给了高，那么子盒子只给宽亦有高度。
  3. 百分比：父元素对于子元素是100%
  4. 在父盒子是作用于直接子集。(只有亲儿子生效)

父元素内部声明使用了弹性布局，使用了弹性布局后子元素的float、clear和vertical-align 属性将失效

弹性布局 布局之后，直接子级沿着主轴排列（主轴水平向右，侧轴垂直向下）

弹性布局默认不会换行

/* 变换为弹性布局 */
display: flex;
/* 换行属性 */
flex-wrap: wrap;
/* 主轴对齐分布属性 */
justify-content;
/* 平均分布 */
space-evenly;
/* 两端分布 */
space-between;
/* 等距分布 */
space-around;
/* 居中 */
justify-content: center;
/* 起点对齐 */
flex-start
/* 终点对齐 */
flex-end
/* 多行侧轴分布对齐属性，其他属性值和主轴一致 */
align-content：space-around
/* stretch 默认值 拉伸 把侧轴平均分，内容顶对齐 */
/* 单行行侧轴分布对齐属性 */
align-items: center

/* 控制子项自己在侧轴的排列方式 */
align-self

.container{

百分比布局

百分比布局又叫流式布局，也称为非固定像素布局

通过盒子的宽度设置成百分比来根据屏幕的宽度来进行伸缩，不受固定像素的限制，内容向两侧填充。

流式布局方式是移动web开发使用的比较常见的布局方式之一。

​ 数值单位 %

​ 几个百分比布局中常见的属性

  • min-width 设置元素的最小宽度, 如果元素的宽度比这个值小，强制按照这个值显示

  • min-height 设置元素的最小高度

  • max-width 设置元素的最大宽度,如果元素的宽度比这个值还大，强制按照这个值显示

  • max-height 设置元素的最大高度


媒体查询

根据屏幕宽度去修改css样式的功能

**语法：基本格式 @media (条件) and (条件) { 条件成立就使用这个css样式 } **

/* min-width最小宽度  max-width最大宽度 */
@media (min-width:641px) and (max-width:980px){
           /* 注意选择器的权重要能覆盖上面的 */
            ul li{
                width: 50%;
            }
    }
       @media (min-width:0px) and (max-width:640px){
            ul li{
                width: 100%;
            }
       }

响应版心

ul{
    /* 最大不能超出1400px */
    max-width: 1400px;
    /* 宽度如果使用百分比宽度会变大或者变小 */
    /* max-width可以限制这个盒子最大不超过多少 */
    /* 小于1400px占80% */
    /* width: 80%; */
    width: auto;
    /* padding: 0 20px; */
    margin: auto;
    display: flex;
    /* 换行属性 */
    flex-wrap: wrap;
}
/* 注意覆盖的写法一定要从大屏覆盖到小屏 大屏在上面 小屏在下面这个顺序 */
@media (max-width:980px){
     /* 注意选择器的权重要能覆盖上面的 */
     ul li{
     width: 50%;
        }
}
@media (max-width:640px){
     ul li{
      width: 100%;
      	}
}
<!-- pc没有书写条件，全部屏幕生效 -->
<link rel="stylesheet" href="css/pc.css">
<!-- 通过给标签添加一个media属性去书写条件 media="(条件)" -->
<!-- max-width:980px小于980px生效 -->
<link rel="stylesheet" href="css/pad.css" media="(max-width:980px)">
<!--max-width:640px小于640px生效 -->
<link rel="stylesheet" href="css/phone.css" media="(max-width:640px)">
